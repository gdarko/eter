# eter

A tiny cross-platform menu-bar internet radio player, built with Python and
PySide6 (Qt 6).

## Run from source

```bash
python3 -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -e ".[dev]"
python -m eter
```

Click the tray icon and pick a station.
